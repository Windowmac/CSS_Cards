CSS Tips

Tips and picked-up formatting tricks to help with CSS styling

Click on a box to select the code

FlexBox Row

Use these properties to create a flexbox row layout

 .row {

        display: flex;

        flex-direction: row;
        
        flex-wrap: wrap;
        
        }

FlexBox Column

Use these properties to create a flexbox column layout

 .column {

        display: flex;
            
        flex-direction: column;
                    
        flex-wrap: wrap;
                    
        }

Flex Shorthand

Use this shorthand on flexed items

 .item {

        flex: 1 1 0; /*grow, shrink, basis/*
                    
        }

Linear Gradient

Create a linear gradient using CSS

 .item {

        background: linear-gradient(color1, color2)
                    
        }

Hover Light-up

change colors of an area on mouse-over to add a light-up effect

 .item:hover {

        background-color: color;
                    
        }

User-Select all

Allow user to select a body of text on-click

 .item {

        user-select: all;
                    
        }
