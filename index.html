CSS Tip Cards

Tips and picked-up formatting tricks to help with CSS styling

FlexBox Row

Use these properties to create a flexbox row layout

.row {

flex-direction: row;

flex-wrap: wrap;

}

FlexBox Column

Use these properties to create a flexbox column layout

.column {

display:flex;

flex-direction:column;

}

CSS Gridstyle

Use these properties to create a CSS Grid layout

.grid {

display:grid;

width:100%

grid-template-columns:

repeat(12, 1fr);

}

Liner Gradient

Create a linear gradient using CSS

.linear-gradient-background {

background-image: linear-gradient-background (

rgba(232, 102, 236, 0.3) 0%

rgba(232, 102, 236, 0.6) 100%

);

}
